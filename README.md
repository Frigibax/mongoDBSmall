# MongoDB Stock Market Simulator & Dashboard

A lightweight, real-time stock market simulation project showcasing how to handle live time-series data using **Python**, **MongoDB**, **Flask**, and **Chart.js**.

This project consists of two parts running in tandem:
1. **The Simulator (`market_simulator.py`)**: A background worker that connects to a local MongoDB instance. It seeds initial prices for major global indexes (like the S&P 500, NIFTY 50, and BSE SENSEX) and then endlessly generates random market fluctuations (-0.5% to +0.5%) every few seconds, storing the exact timestamps and prices.
2. **The Web Dashboard (`app.py`)**: A Flask-based REST API and stunning dark-mode web interface. It fetches the latest data history from the database and visualizes it dynamically on the frontend without refreshing the page.

## Prerequisites

To run this project anywhere, you need the following installed on your machine:
* **[Python 3.x](https://www.python.org/downloads/)**
* **[MongoDB](https://www.mongodb.com/try/download/community)** (Make sure the MongoDB server is running locally on the default port `27017`)
* **Git** (to clone the repository)

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Frigibax/mongoDBSmall.git
   cd mongoDBSmall
   ```

2. **Create and activate a virtual environment:**
   *On Windows (PowerShell):*
   ```powershell
   python -m venv venv
   .\venv\Scripts\activate
   ```
   *On macOS / Linux:*
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

3. **Install the required Python dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

## How to Run

To see the live action, you will need to open **two** separate terminal windows. Make sure your virtual environment is activated (`.\venv\Scripts\activate`) in both windows!

### Terminal 1: Start the Data Simulator
This script populates the database and continually adds live trades.
```bash
python market_simulator.py
```
*(Leave this running in the background).*

### Terminal 2: Start the Web Server
This serves the frontend dashboard to your browser.
```bash
python app.py
```
Once the server starts, open your web browser and go to:  
**http://127.0.0.1:5000**

You will see the live, animating Chart.js dashboard tracking the data generated by the simulator!
